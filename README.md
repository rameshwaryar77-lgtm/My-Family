# My Family Website

A responsive family website to share members, values, and photo memories.

## Links

- Repository: https://github.com/rameshwaryar77-lgtm/My-Family
- Live Website: https://rameshwaryar77-lgtm.github.io/My-Family/

## Project Structure

```text
My family/
|-- index.html
|-- photos.html
|-- .nojekyll
|-- README.md
`-- Photos/
```

## Features

- Responsive design for desktop and mobile
- Family intro and values sections
- Dedicated photo gallery page
- Social profile links
- GitHub Pages deployment workflow

## Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/rameshwaryar77-lgtm/My-Family.git
   cd My-Family
   ```
2. Open `index.html` in a browser.

## Deployment

The site deploys automatically from `main` using GitHub Actions.

If the live link shows 404, confirm:
- Repository visibility is Public
- Settings -> Pages -> Source is set to GitHub Actions
- Latest workflow run is successful
